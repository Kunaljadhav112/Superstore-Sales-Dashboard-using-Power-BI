# Superstore-Sales-Dashboard-using-Power-BI
Superstore Sales Dashboard using Power BI

## 🎓 Skills Demonstrated

- Power BI Dashboard Development  
- Data Cleaning and Transformation using Power Query  
- Data Modeling and Relationship Building  
- DAX (Data Analysis Expressions) for KPI Calculation  
- Sales and Profit Analysis  
- Data Visualization and Interactive Dashboard Design  
- Business Intelligence and Performance Reporting  
- Trend Analysis and Business Insights Generation  

---

## 📊 Business Impact

This project provides actionable business insights to improve sales performance and strategic decision-making:

- Identified top-performing products, categories, and regions to optimize sales strategy  
- Enabled tracking of key metrics such as Total Sales, Profit, and Order Volume  
- Highlighted loss-making products and regions to reduce financial risk  
- Improved visibility into monthly and yearly sales trends for better forecasting  
- Helped optimize regional and category-level performance  
- Supported data-driven decision making using interactive Power BI dashboards
