# 🛒 Super Store Sales Dashboard (Power BI)

An interactive and insight-driven **Super Store Sales Dashboard** built using **Power BI** to analyze sales performance, profitability, delivery efficiency, and customer behavior across multiple regions and categories.

This dashboard enables business stakeholders to monitor KPIs, identify high-performing segments, and make data-driven decisions to optimize revenue and profitability.

---

## 🚀 Project Overview

The **Super Store Sales Dashboard** provides a comprehensive analysis of retail sales performance across regions, segments, categories, and shipping modes.

It transforms raw transactional data into visually intuitive insights that support:

- Revenue growth analysis
- Profit trend monitoring
- Customer segmentation evaluation
- Shipping performance assessment
- Regional sales comparison

---

## 📊 Executive KPIs

The dashboard highlights key performance indicators:

- **Total Sales:** 1.57M
- **Total Quantity Sold:** 22.32K
- **Total Profit:** 175.26K
- **Average Delivery Time:** 3.93 Days

These KPIs provide a quick snapshot of overall business health.

### 2️⃣ Sales by Payment Mode
